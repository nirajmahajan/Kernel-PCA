# Kernel PCA

Scholkopf, B., & Smola, A., & Muller, K. (1996) Nonlinear Component Analysis as a Kernel Eigenvalue Problem

Kwok, J., & Tsang, I. (2003) The Pre-Image Problem in Kernel Methods
